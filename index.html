First a Triangle

a 100 days project to create a VJ program

So here's the plan. I'm going to make super basic piece of VJ software using web technologies that will run in a browser. I'll focus on basic geometric shapes and the interesting things you can do with them in HTML/CSS/SVG/JS/etc.

I've always like minimal visuals in a rave so this is a chance to see how far I can push things using basic primitives, and whether by adding one effect every day I end up with something I'd actually be happy to run on a big screen in a dance.

  • day 1: a triangle
  • day 2: fading out triangles
  • day 3: horizontal/vertical cyan/red shaded triangles
  • day 4: diagonal cyan/red shaded triangles
  • day 5: pulsing rectangle
